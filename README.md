# 🏬 Superstore Sales & Profit Analytics Dashboard

## 📊 Overview

The Superstore Sales & Profit Analytics Dashboard is an interactive Tableau project that provides insights into sales performance, profitability, customer behavior, and regional business trends across the United States. The dashboard enables stakeholders to monitor key business metrics and identify opportunities for growth and operational improvement.

---

## 🎯 Project Objectives

* Track overall sales and profitability performance
* Analyze customer segments and purchasing patterns
* Evaluate product category and sub-category performance
* Identify top-performing states and regions
* Monitor sales trends over time
* Support strategic business decision-making

---

## 🛠️ Tools & Technologies

* Tableau Desktop
* Calculated Fields
* Parameters & Filters
* Geographic Mapping
* Interactive Dashboards
* Excel Dataset

---

## 📂 Dataset Information

The analysis is based on the Sample Superstore Dataset containing retail transaction records from multiple regions across the United States.

### Key Data Attributes

* Order Date
* Sales
* Profit
* Customer Segment
* Category
* Sub-Category
* Region
* State
* Returns
* Product Details
